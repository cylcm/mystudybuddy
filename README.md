# 📚 MyStudyBuddy

**Smart study companion for Primary 5–6 & Secondary 1–5**

A fully portable, AI-powered study app that runs in any browser — no installation needed.

## Features
- 🧠 Smart Quiz (Primary 5–6, Secondary 1–5, all subjects)
- ✍️ Grammar & Spelling Coach
- 🔀 Word Scramble Challenge
- 🈳 Language Corner (Chinese, Japanese, French, German)
- 💡 Explain It Simply (with AI diagrams)
- 📅 Study Plan
- 📊 My Scores

## Setup
1. Open the app URL in Safari on iPhone/iPad
2. Click the 🔑 API Key button (top-right) and enter your [Anthropic API key](https://console.anthropic.com)
3. Click ⚙️ to set up your profile
4. Start studying!

## Add to Home Screen (iPhone/iPad)
1. Open in Safari
2. Tap Share → Add to Home Screen
3. Works like a native app!
