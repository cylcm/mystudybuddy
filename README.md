# 📚 MyStudyBuddy v5.3
**Smart Study Companion on the GO ✈️**  
AI-powered educational suite for Singapore students K1–Secondary 5.  
Single-file standalone HTML app — no build step, no backend, no server required.

---

## 🚀 How to Launch MyStudyBuddy

### Option 1 — Live Web App (Recommended)
Just open your browser and visit:
```
https://cylcm.github.io/mystudybuddy/
```
No installation needed. Works on desktop, tablet, and mobile.

---

### Option 2 — Run Locally (Offline)
1. Download `index.html` from this repo
2. Open the file directly in your browser:
   - Double-click `index.html`, **or**
   - Drag and drop it into Chrome/Safari/Edge
3. The app runs fully in your browser — no server needed

---

### Option 3 — Install as an App (PWA)
**On Desktop (Chrome):**
1. Visit the live URL above
2. Click the **install icon** (⊕) in the address bar
3. Click **Install** — it appears as a desktop app

**On iPhone/iPad (Safari):**
1. Visit the live URL
2. Tap the **Share** button → **"Add to Home Screen"**
3. Tap **Add** — it appears as an app icon

**On Android (Chrome):**
1. Visit the live URL
2. Tap the **three dots menu** → **"Add to Home Screen"**

---

## 🔑 First-Time Setup — API Key Required

MyStudyBuddy uses the **Anthropic Claude API** to power AI features (quizzes, grammar check, explanations, etc.).

1. Get your **free API key** from [console.anthropic.com](https://console.anthropic.com)
2. Open the app → click the **🔑 key icon** (top-right corner)
3. Paste your API key → click **Save**
4. Your key is stored only in your browser — never shared

> ⚠️ Without an API key, offline features (Sudoku, Study Planner, Vocabulary Match) still work, but AI features will be disabled.

## 🌐 Enable GitHub Pages (First Time Only)

If the live URL above doesn't work yet, enable GitHub Pages:

1. Go to your repo → **Settings → Pages**
2. Under **Source**, select **"Deploy from branch"**
3. Branch: `main` | Folder: `/ (root)`
4. Click **Save**
5. Wait 1–2 minutes, then visit:  
   `https://cylcm.github.io/mystudybuddy/`
